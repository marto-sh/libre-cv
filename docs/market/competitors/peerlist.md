> **Founders:** Abhijay Jain, Akash Goel

## Company Overview

Peerlist is a professional networking platform designed specifically for tech professionals. It focuses on creating "proof-of-work" profiles that showcase actual contributions rather than just job titles and descriptions. The platform integrates with GitHub, Dribbble, and other work platforms to automatically populate profiles with real projects and contributions.

Peerlist positions itself as an alternative to LinkedIn for the tech community, emphasizing authenticity and real work over polished resumes.

## Product/Service Offering

### Core Features:
- **Proof-of-Work Profiles:** Automatically pulls contributions from GitHub, Dribbble, and other platforms
- **Skill Verification:** Skills are verified through actual code contributions and projects
- **Community Engagement:** Focus on meaningful professional interactions
- **Job Opportunities:** Curated tech job listings
- **Portfolio Showcase:** Highlight open-source contributions and personal projects

### Unique Aspects:
- **GitHub Integration:** Deep integration shows actual code contributions, repositories, and activity
- **No Traditional Resume:** Profiles focus on what you've built, not just where you've worked
- **Tech-Focused:** Designed specifically for developers, designers, and tech professionals
- **Community-Driven:** Emphasis on peer recognition and collaboration

**USP:** "Your work speaks for itself" - Peerlist creates professional profiles based on actual contributions rather than self-reported achievements.

## Market Perception

### Strengths

- **Authenticity:** Real work verification creates more trustworthy profiles
- **Tech-focused:** Designed specifically for the needs of tech professionals
- **Automatic updates:** Profiles stay current through platform integrations
- **Community-driven:** Strong focus on peer recognition and collaboration
- **Modern approach:** Moves beyond traditional resume formats

### Weaknesses

- **Limited reach:** Much smaller network compared to LinkedIn
- **Niche focus:** Primarily useful for tech professionals only
- **Integration limitations:** Currently limited to GitHub and a few other platforms
- **Discovery challenges:** Smaller user base means fewer networking opportunities
- **No agent capabilities:** Purely a profiling and networking platform

Peerlist validates the demand for more authentic, work-based professional profiles in the tech industry. However, it remains a passive profiling platform without any autonomous capabilities. libre-cv takes this concept further by:

1. **Expanding beyond tech** to serve professionals across all industries
2. **Adding autonomous agent capabilities** that can actively represent professionals
3. **Providing data ownership** through self-hosted, open-source architecture
4. **Creating structured, queryable identities** rather than just display profiles

Peerlist shows that professionals want more than just traditional resumes, but libre-cv aims to fulfill this need with a comprehensive, autonomous identity framework.
