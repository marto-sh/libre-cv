# Competitor Landscape

## Direct Competitors

Platforms where an AI agent actively represents professionals — finding opportunities, screening interactions, or applying on their behalf.

| Competitor | Model | Key Differentiator |
|---|---|---|
| [Jack & Jill AI](jack_and_jill_ai.md) | Proprietary, commission-based (10% of first-year salary) | Dual-agent system: "Jack" for seekers, "Jill" for employers. $20M seed, ~50k users. |
| [Jobright](jobright.md) | Proprietary SaaS | "Autopilot" job search — agent finds, customizes, and submits applications autonomously. |
| [Maayu](maayu.md) | Proprietary SaaS | Fully autonomous AI assistant with job matching and auto-apply. |
| [Careery](careery.md) | Proprietary SaaS | Autonomous agent that searches and applies 24/7 at scale. |

## Professional Profile & Portfolio Platforms

Rich professional presence beyond a traditional resume, but without an autonomous agent acting on your behalf.

| Competitor | Model | Key Differentiator |
|---|---|---|
| [LinkedIn](linkedin.md) | Proprietary, ad/subscription-funded | The incumbent. Massive network, but passive profiles and no data portability. |
| [Peerlist](peerlist.md) | Proprietary, free tier | Proof-of-work profiles for tech professionals. Pulls from GitHub, Dribbble, etc. |
| [Polywork](polywork.md) | Shut down (Jan 2025) | Multi-faceted identity beyond job titles (timeline, badges). Validates the problem space. |

## Resume Builders & Structured Data

Tools that structure professional data, but treat it as a formatted document rather than a living, interactable identity.

| Competitor | Model | Key Differentiator |
|---|---|---|
| [JSON Resume](json_resume.md) | Open source (MIT) | Open JSON schema standard for resume data. Portable, versionable, community-driven. |
| [Reactive Resume](reactive_resume.md) | Open source, self-hosted | Privacy-first resume builder with API access. Docker-deployable. |
| [OpenResume](open_resume.md) | Open source | Browser-based builder and parser. Data stays local. |
| [FlowCV](flowcv.md) | Proprietary, freemium | Polished resume builder with modern templates. |

## Open Source Job Application Bots

Automation tools focused on bulk-applying rather than identity ownership.

| Competitor | Model | Key Differentiator |
|---|---|---|
| [AIHawk](aihawk.md) | Open source (AGPL) | AI bot that automates LinkedIn job applications. No identity/profile layer. |

## Where libre-cv Stands

No existing competitor combines all three pillars of libre-cv:

1. **Structured, queryable professional identity** (like JSON Resume)
2. **Autonomous agent that acts on your behalf** (like Jobright, Jack & Jill)
3. **Libre philosophy — own your code and your data** (like Reactive Resume)

The closest competitors each cover one or two of these dimensions, leaving the intersection unoccupied.
